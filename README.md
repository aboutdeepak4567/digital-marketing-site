# Creative Agency – Digital Marketing Site 🚀

A modern, high-performance portfolio website for a digital marketing agency, completely rebuilt with **Vue 3** and **Vite**. Features interactive 3D UI elements, responsive design, CSS scroll animations, and a sleek dark-mode-inspired aesthetic.

🌍 **Live Demo:** [Visit the Website](https://digital-marketing-site-ten.vercel.app/)

## ✨ Key Features
- **Modern Tech Stack:** Migrated from vanilla HTML/JS to a component-based **Vue 3** (Composition API) architecture.
- **Vite Setup:** Lightning-fast HMR and optimized production builds.
- **Interactive UI:**
  - 3D mouse-tilt parallax effects on hero and project cards.
  - Custom scroll-spy navigation.
  - Floating badge animations.
  - Smooth fade-in-on-scroll using `IntersectionObserver`.
- **Dynamic Components:**
  - Auto-rotating testimonials carousel.
  - Animated counting stats (using `nextTick` synchronization).
  - Fullscreen project lightbox modal.
  - Reactive, client-side validated contact form.

## 🛠️ Built With
- **Vue 3** (Composition API, `<script setup>`)
- **Vite 6**
- **Vanilla CSS3** (CSS Variables, Flexbox, Grid, Animations)

## 🚀 Getting Started

To run this project locally, simply clone the repository and install the dependencies:

```bash
# Clone the repository
git clone https://github.com/aboutdeepak4567/digital-marketing-site.git
cd digital-marketing-site

# Install dependencies
npm install

# Start the Vite development server
npm run dev
```

Visit `http://localhost:5173` in your browser to view the site.

## 📦 Production Build

To generate the production-ready static assets in the `/dist` folder:

```bash
npm run build
```
